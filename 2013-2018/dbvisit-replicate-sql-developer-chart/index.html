Dbvisit replicate – SQL Developer chart

Here is a quick SQL Developer report which display a chart about the Dbvisit replicate lag over the last hours The idea is to have the following chart showing the lag in MINE and APPLY processes. Here is an example where I stopped the replication to show some lag.

The query is on the DBVREP.DBRSCOMMON_LAG_STATS on the APPLY side, which display the wallclock time with timestamp from the MINE and from the APPLY.
